Seat Functions

AddSeat(name, pos, ang) [server]
Add a seat to the vehicle. The first seat is always the pilot.

Parameters:

  • name The name of the seat for easy reference
  • pos The position of the seat in local coordinated
  • ang The angles of the seat in local angles

Returns:

    The seat entity itself for convenience
GetSeat(index) [shared]
Retrieve an actual seat entity.

Parameters:

  • index The index of the seat. Can be a number or string.

Returns:

    The found entity or NULL
